## 💻 How to Run (Quickstart)

```bash
git clone https://github.com/Sudharsan6565/HITCLOUD.git && cd HITCLOUD
go build -o agent47 main.go
./agent47
```

> Deploy via Docker, ECS, Lambda, or bare metal. Agent registers and scans every 30s.
